ขั้นตอนการเปิดใช้งานบริการ NT Cloud Service
ฝ่ายงาน / ผู้ขอใช้บริการ
Helpdesk
Support
START
1
กรอกข้อมูลคำขอใช้บริการ กรอกแบบฟอร์ม NT Cloud Service ตามลิงก์ที่กำหนด
2
ส่งเอกสารและรายละเอียด แนบเอกสารใบเชื่อมโยง ใบเสนอราคา และ Key เข้าสู่ระบบ OM
3
ตรวจสอบเอกสาร ตรวจสอบข้อมูลเอกสาร ใบเชื่อมโยง และใบเสนอราคา
4
กรอกแบบฟอร์มเปิดบริการ บันทึกข้อมูลในแบบฟอร์มการเปิดบริการ NT Cloud
5
สร้าง VM ดำเนินการสร้าง VM ตามข้อมูลในแบบฟอร์ม
6
แจ้งข้อมูลเข้าใช้งาน ส่ง Username และ Password ไปยังอีเมลลูกค้า
7
ตอบกลับผลการดำเนินงาน อัปเดตข้อมูลในแบบฟอร์มการเปิดบริการ NT Cloud
8
ตรวจสอบผลจาก Support ตรวจสอบความครบถ้วนของข้อมูลในแบบฟอร์ม
9
ตอบกลับข้อมูลให้ฝ่ายงาน ส่งรายละเอียดผ่านอีเมลหรือลิงก์ NT Cloud
10
บันทึกรายละเอียดใน OM นำข้อมูลที่ได้รับไปใส่ในระบบ OM เพื่อเรียกชำระเงินลูกค้า
11
ตรวจสอบการเรียกชำระเงิน ยืนยันว่าระบบ OM ดำเนินการครบถ้วน
12
ปิด Task งาน สรุปผลและปิดงานในระบบ
END
ส่งข้อมูลไปที่ ntcloud@ntplc.co.th แจ้งเปิดงานผ่าน support_cloud@ntplc.cloud
